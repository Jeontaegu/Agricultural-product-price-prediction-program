🥬 농산물 가격 예측 알리미 (Agri-Price Predictor)

기후 변화와 물가 상승으로 장보기가 고민되는 일반 소비자를 위한 모바일 웹 기반 농산물 가격 예측 서비스입니다.
과거 3년 치의 농산물 가격 데이터, 주산지의 날씨(기온, 강수량 등), 재배 면적 데이터를 AI 모델로 분석하여 내일의 농산물 소매 가격을 직관적으로 알려줍니다.

##스크린샷


🎯 기획 배경 및 타겟 사용자

문제 정의: 기상 악화(태풍, 폭우 등)나 작황 부진으로 농산물 가격이 급등락하여 소비자의 장바구니 부담이 가중됨.

타겟 사용자: 합리적인 시기에 식자재를 구매하고자 하는 일반 소비자 및 1인 가구.

목표: 복잡한 통계 수치 대신, 직관적인 UI와 '소비자 가이드'를 통해 오늘 살지, 내일 살지 결정하는 데 도움을 줌.

✨ 주요 기능 (Features)

주요 농산물 4종 예측 지원: 배추, 무, 마늘, 사과의 소매 가격 예측

AI 가격 예측 안내: '오늘의 실제 가격'과 AI가 예측한 '내일의 가격'을 비교하여 등락폭 제공

소비자 맞춤형 구매 가이드: 가격 변동(상승/하락) 트렌드에 따른 구매 추천 멘트 및 색상 심리학을 적용한 UI 제공 (예: 가격 하락 시 파란색/스마일, 가격 상승 시 빨간색/경고)

주산지 날씨 연동 정보: 각 농산물별 주요 산지의 최근 기상 정보를 직관적인 아이콘과 함께 제공

직관적인 시각화 (Chart): 최근 14일간의 실제 가격 추이(실선)와 내일 예측 가격(점선)을 하나의 꺾은선 그래프로 제공하며, 그라데이션 효과로 시각적 완성도 향상

API 키 관리 (클라이언트): 사용자 개인이 발급받은 KAMIS 및 기상청 API 키를 브라우저 로컬 스토리지에 안전하게 저장하고 관리하는 설정 메뉴 제공

🛠️ 기술 스택 (Tech Stack)

Frontend (현재 구현됨 - agri_price_predictor_v2.html)

HTML5 / CSS3 / JavaScript (Vanilla)

Tailwind CSS: 모바일 뷰포트 고정 및 빠른 반응형 UI 스타일링

Chart.js: 과거 추이 및 예측 가격 시각화 (꺾은선 그래프)

FontAwesome: 직관적인 아이콘 제공

Backend & AI (데이터 수집용 파이썬 스크립트 제공됨 - data_collector.py)

Language: Python 3.9+

Framework (예정): FastAPI (예측 결과를 프론트엔드에 전달하기 위한 REST API 구축)

AI/ML (예정): LightGBM, XGBoost, 또는 Prophet (시계열 데이터 기반 가격 예측 모델링)

Data Processing: Pandas, requests, lxml (웹 스크래핑 방식 활용 시)
