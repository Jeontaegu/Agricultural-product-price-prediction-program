# 🌾 농산물 가격 예측 알리미 (Agricultural Product Price Predictior)

## 📌 프로젝트 개요 (Project Overview)
기후 변화와 물가 상승으로 장보기가 고민되는 일반 소비자를 위한 모바일 웹 기반 농산물 가격 예측 서비스입니다. 주요 농산물(배추, 무, 깐마늘, 사과 등)의 실시간 소매 가격과 익일 가격 변동 추이를 분석하고 소비자에게 합리적인 구매 가이드를 제공하는 **모바일 최적화 웹 애플리케이션**입니다.

## 🎯 1. 어떤 문제를 해결하려 했는지 (Problem Statement)

소비자들은 마트나 전통시장에서 장을 볼 때 채소나 과일 가격이 매일 급등락하는 현상을 자주 겪습니다. 하지만 일반 소비자가 농산물 가격의 변동 요인(기상 악화, 주산지 작황, 출하량 변화 등)을 실시간으로 파악하고 언제 사는 것이 가장 유리한지 판단하기는 매우 어렵습니다. 기존의 농산물 정보 조회 서비스들은 다음과 같은 한계가 있었습니다.

*   **과거 데이터 중심의 단순 나열:** 단순한 과거 시세 표나 통계 수치만 제공할 뿐, "내일 가격이 오를지 내릴지"에 대한 직접적인 예측 인사이트를 주지 못합니다.
*   **복잡한 정보 접근성:** 기상청의 주산지 날씨 정보와 KAMIS의 유통 가격 정보가 분산되어 있어, 소비자가 연관 지어 분석하기 번거롭습니다.
*   **직관적인 구매 타이밍 안내 부재:** 주부나 1인가구 소비자가 "오늘 사야 할지, 내일로 미뤄야 할지" 결정을 내릴 수 있는 실질적인 가이드라인이 부족했습니다.

**💡 해결 목표:**
본 프로젝트는 **기상 정보와 유통 가격 데이터를 융합한 AI 예측 결과**를 바탕으로, 소비자에게 내일의 예상 가격과 증감 폭을 명확하게 제시하고 **"구매를 미루세요"** 혹은 **"오늘 미리 구매하세요"**와 같은 직관적인 행동 지침(Actionable Guide)을 제공하여 가계 경제에 실질적인 도움을 주는 것을 목표로 하였습니다.

## 2. 어떤 기능을 구현했는지 (Implemented Features)

모바일 앱 형태의 미려한 UI/UX(Tailwind CSS)와 반응형 차트 라이브러리(`Chart.js`)를 도입하여 다음과 같은 핵심 기능을 구현했습니다.

### 🥬 1. 주요 품목별 실시간 탭 네비게이션
*   장바구니 물가에 영향이 큰 핵심 농산물(**배추, 무, 마늘, 사과**)을 직관적인 탭 버튼으로 즉시 전환하여 조회할 수 있습니다.
*   품목을 바꿀 때마다 백엔드 설정 데이터(`pythonConfig`)와 연동되어 주산지 위치, 날씨, 가격 트렌드가 다동으로 갱신됩니다.

### 🤖 2. AI 기반 익일 가격 예측 및 증감 시각화
*   오늘의 실제 가격과 AI 모델이 예측한 **내일의 예상 소매 가격**을 비교하여 보여줍니다.
*   가격 변동 폭(오름세/내림세)에 따라 소비자 친화적인 색상 코드(하락 시 파란색/상승 시 빨간색)와 아이콘을 동적으로 적용하여 시인성을 극대화했습니다.

### 🌦️ 3. 주산지 기상 정보 실시간 연동 표시
*   가격 변동의 핵심 원인인 주산지(예: 배추-대관령, 무-서귀포 등)의 최근 기상 상태(강수량, 태풍 특보, 일조량 등)를 카드 하단에 노출하여, 사용자가 가격 변동의 맥락을 자연스럽게 이해할 수 있도록 설계했습니다.

### 📈 4. 2주간 가격 추이 및 예측 구간 시각화 차트 (`Chart.js`)
*   최근 14일간의 과거 가격 데이터와 익일 AI 예측 가격을 하나의 연속된 라인 차트로 시각화합니다.
*   특히 **오늘과 내일(AI 예측) 사이의 구간을 보라색 점선(`segment`)으로 구분**하여, 사용자가 과거 흐름과 미래 예측치를 직관적으로 구분할 수 있게 구현했습니다.
*   툴팁(Tooltip)을 통해 마우스 오버 시 정확한 원화 단위의 가격 정보를 확인할 수 있습니다.

### 💡 5. 소비자 맞춤형 행동 가이드 (Action Guide)
*   가격 상승/하락 여부에 따라 **"구매를 내일로 미루세요!", "오늘 미리 구매하시는 것이 유리합니다!"** 등 명확한 구매 타이밍 지침을 대형 카드 형태로 강조하여 제공합니다.
*   친근한 이모지와 공감 가는 설명 텍스트를 통해 복잡한 유통 정보를 일반 소비자도 쉽게 이해하도록 구성했습니다.

### ⚙️ 6. Open API 키 설정 모달 (Settings & Sync Status)
*   사용자가 직접 KAMIS API 키와 기상청 API 키를 입력하고 브라우저 로컬 스토리지(`localStorage`)에 안전하게 보관할 수 있는 설정 모달을 제공합니다.
*   키 설정 여부에 따라 상단 상태바의 동기화 핑(Ping) 애니메이션과 시스템 상태 문구가 실시간으로 변경되어 시스템 연동 상태를 투명하게 알려줍니다.
